# Memorial Descritivo

Utilize este diretório para o texto que explica os critérios técnicos (materiais, cálculos estruturais, dimensionamento elétrico, implantação). Proposta de fluxo:
1. Redija no editor de preferência (Word/Pages) e salve a fonte aqui.
2. Exporte para PDF para circulação.
3. Anote no cabeçalho do documento o vínculo com as referências normativas arquivadas ao lado.
