# Desenhos e Modelos

Armazene plantas, cortes, modelos CAD/BIM e renders. Sempre incluir:
1. Formato editável (DWG, DXF, SKP, etc.).
2. Exportação em PDF com legenda atualizada (data, responsável, versão).

Organize subpastas por disciplina (`eletrica/`, `civil/`, etc.) se o volume crescer. Documente revisões na legenda e atualize o `inventario_arquivos.csv`.
