# Entrada de Requisitos

Use esta pasta para armazenar:
- briefings operacionais (ex.: demandas do BPM, mapas de incidência);
- checklists de órgãos reguladores;
- levantamentos elétricos e civis coletados in loco.

Formato sugerido:
1. Nomeie arquivos como `YYMMDD_local_tipo.ext` (ex.: `251110_campo_grande_briefing.docx`).
2. Mantenha um índice simples no topo deste README listando os documentos adicionados e sua relação com o restante do projeto técnico.

Planilhas disponíveis:
- `locais_instalacao.csv` – template para registrar AISP/BPM, coordenadas e status de cada ponto de instalação.
