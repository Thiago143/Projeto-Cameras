# Imagens dos Postes

Quando as fotos/croquis estiverem disponíveis:
- Nomeie os arquivos como `local_data_perspectiva.jpg` (ex.: `bangu_20251112_frontal.jpg`).
- Registre no topo deste README a lista de imagens com coordenadas aproximadas e observações (altura, interferências, estado do poste).
- Remova metadados sensíveis com `sips --stripImageMetadata <arquivo>` antes de compartilhar.
