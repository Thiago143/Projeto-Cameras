# Base 2025-11

A apresentação `PROJETO_DE_IMPLEMENTAÇÃO_DE_CÂMERAS_OSTENSIVAS-1.pdf` foi validada como referência mais recente do projeto e deve orientar os demais artefatos até que novas versões sejam homologadas.

Quando incorporar atualizações:
1. Armazene a fonte editável (Keynote/PowerPoint) neste mesmo diretório junto ao PDF exportado.
2. Registre as mudanças relevantes no `inventario_arquivos.csv` marcando `status=valido` para o novo conjunto e `status=substituir` para versões antigas.
3. Caso a apresentação gere entregáveis derivados (ex.: briefings táticos), abra subpastas nomeadas por sprint ou data (`base/2025-11/sprint-01`) e documente o relacionamento em cada README.
