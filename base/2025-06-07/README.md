# Base 2025-06-07

Material de transição criado entre junho e julho/2025. Considere estes arquivos (`sumario_executivo_cameras.pdf` e o brasão atualizado) como o ponto de partida para a “roupagem nova”.

Boas práticas:
- Use o brasão desta pasta como referência oficial e registre qualquer nova versão (SVG, AI, etc.) aqui antes de exportar para outros formatos.
- Quando atualizar o sumário executivo, gere sempre o par fonte + export (ex.: `.docx` + `.pdf`) e mantenha apenas a versão vigente neste diretório.
- Atualize o `inventario_arquivos.csv` marcando `status=valido` ou `status=substituir` assim que uma revisão for aprovada.
