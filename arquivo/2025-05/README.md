# Arquivo 2025-05

Este diretório concentra o acervo original entregue em maio/2025 (capturas, fontes editáveis e PDFs exportados). Todos os itens estão marcados como `atualizar` ou `substituir` no `inventario_arquivos.csv`, então use-os apenas como referência histórica.

Fluxo sugerido:
1. Compare cada fonte (`.docx`/`.pages`) com a versão vigente e edite uma cópia em `base/` ou no novo layout antes de republicar.
2. Documente na coluna `referencia_substituta` do inventário quando houver uma versão nova definitiva.
3. Após homologar a nova versão, mova o arquivo legado para um subdiretório de arquivo frio (`arquivo/2025-05/legado/`) ou compacte-o, mantendo este diretório apenas com o mínimo necessário para rastreabilidade.
